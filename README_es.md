# sec_installer
Este script (python) automatiza la instalacion de herramientas comunes para realizar tests de seguridad (Pentesting).

Podras elegir entre:

- Instalar todas las herramientas disponibles en el script.
  
- Installar una seleccion mia.
  
- Elegir una Categoria a instalar.
  
- Seleccionar una o mas herramientas de interes.


### Uso
Instalar previamente python3, pip, git y go:

```bash
sudo apt update && sudo apt install -y python3 python3-pip git golang
```
Clona el repositorio:
```bash
git clone https://github.com/Exbinary/sec_installer
```
```bash
cd sec_installer
```
Ejecuta el script con python3:
```bash
python3 sec_installer.py
```
